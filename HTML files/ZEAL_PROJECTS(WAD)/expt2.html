EXPERIMENT 2 – WEB PAGE FORMATTING

Technology has transformed the way we live and learn. The internet allows people to access information instantly, communicate globally, and collaborate on innovative projects. Learning HTML and CSS is a fundamental skill for anyone interested in web development. By practicing simple paragraph formatting, we can make content more readable and visually appealing on web pages.

A well-formatted paragraph not only looks neat but also improves user engagement. Using bold, italic, and underlined text, along with horizontal rules ( "hr"), helps divide content clearly. Adding subtle details with formatting tags can guide readers through important points and enhance the overall presentation.
