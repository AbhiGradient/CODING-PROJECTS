EXPERIMENT 3 – TEXT FORMATTING TAGS

Bold text and strong text emphasize important content. Italic text and emphasized text can be used for subtle emphasis.

Underlined text highlights key points. Strikethrough or deleted text shows corrections or removed content.

Computer text can be shown using teletype text or inline code. These are useful for programming examples.

Mathematical or scientific notation can use subscript and superscript: H2O is water and E = mc2 is Einstein's famous equation.
